## 本周完成工作

### 1. 后端项目初始化

- **项目搭建**：使用 Node.js + Express 框架初始化项目
- **技术栈配置**：
  - TypeScript 5.3.2
  - MySQL 8.0+ 数据库
  - Sequelize 7.6.3 ORM 框架
  - JWT 9.0.2 认证令牌
  - Multer 1.4.5 文件上传
  - Express Validator 7.0.1 数据验证
  - dotenv 16.3.1 环境变量管理

### 2. 目录结构设计

- **核心目录**：
  - `src/config/`：配置文件（CORS、数据库连接、环境变量）
  - `src/controllers/`：控制器（认证、素材、订单、收益）
  - `src/middleware/`：中间件（认证、错误处理）
  - `src/models/`：数据模型（用户、素材、标签、订单、收益）
  - `src/routes/`：路由定义
  - `src/services/`：业务逻辑
  - `src/utils/`：工具函数（JWT、密码、文件上传）
  - `public/uploads/`：上传的文件

### 3. 核心功能实现

- **用户认证**：注册、登录、JWT 令牌生成与验证
- **素材管理**：上传、列表查询、详情获取、更新、删除
- **订单管理**：创建订单、订单列表、详情查询、状态更新
- **收益管理**：收益统计、明细查询
- **文件上传**：支持素材文件和缩略图上传
- **数据验证**：请求数据验证和错误处理

### 4. 数据库设计与实现

- **数据库结构**：设计并实现 7 个核心表

  - `material_tags`：素材标签关联表
  - `orders`：订单表
  - `order_items`：订单详情表
  - `earnings`：收益表
- **关系映射**：使用 Sequelize ORM 实现表关联
- **数据迁移**：自动创建和更新表结构
