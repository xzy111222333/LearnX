## 技术难点与解决方案（规划）

### 1. 响应式多终端适配
- **难点**：需要同时适配 PC 端、平板端和移动端
- **解决方案**：使用 Flexbox + Grid + 媒体查询，采用移动优先设计策略

### 2. 文件上传与预览
- **难点**：大文件上传、进度显示、PDF/Word 预览
- **解决方案**：使用分片上传，结合 Element Plus Upload 组件，集成 PDF.js 预览

### 3. 支付流程状态管理
- **难点**：支付状态同步、订单轮询、异常处理
- **解决方案**：使用 WebSocket 实时通知 + 定时轮询双重保障

### 4. 性能优化
- **难点**：页面加载速度、首屏渲染时间
- **解决方案**：图片懒加载、代码分割、组件缓存、CDN 加速
