## 遇到的问题和解决

1. 问题：接口文档、前端调用路径和后端实际实现之间容易出现字段名或目录结构不一致的问题。
   解决：对照 `api.yaml`、前端 API 调用层和后端路由逐项核对，保证核心接口名称、参数含义和响应结构保持一致。

2. 问题：后端接口虽然已经实现，但本地运行时还需要确认数据库连接和接口可访问性，否则截图材料没有说服力。
   解决：完成本地环境核查，确认 MySQL 连接正常、后端接口能够返回数据、前端页面能够正常请求 API。
